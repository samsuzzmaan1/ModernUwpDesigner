## 📌 What is ModernUwpDesigner?

ModernUwpDesigner is a tool for designing XAML interfaces. It works with Modern UWP (Universal Windows Platform) projects using .NET 9 or higher. This helps you build and visualize app layouts before coding or testing on devices.

This tool is aimed at users working with Microsoft Visual Studio or similar. It focuses on creating clean, modern interfaces for Windows apps. You do not need programming skills to use it for previewing or arranging UI elements.

## ⚙️ Opening Your UWP Project

1. Open ModernUwpDesigner.

2. Click on `File` in the top menu and select `Open Project`.

3. Browse to the folder containing your UWP project files.

4. Select the project file ending with `.csproj` for your app.

5. The designer will load your project and display the main XAML file.

6. Use the left panel to select different XAML pages if your project has multiple layouts.

## 🎨 Designing with ModernUwpDesigner

- Use drag-and-drop to add controls like buttons, text boxes, and images.

- Adjust properties in the right panel to change colors, sizes, fonts, and other settings.

- The central canvas shows a live preview of your app interface.

- Use zoom and pan controls to inspect details.

- The tool supports grid layouts, margins, and alignment guides to help position items precisely.

- Changes automatically update in the preview.

---

## 💾 Saving and Exporting Your Work

- After designing, save your project by clicking `File > Save` or pressing `Ctrl + S`.

- The tool updates the XAML files in your project folder directly.

- You can now open your project in Visual Studio and continue coding with the new interface.

- ModernUwpDesigner does not change code logic, only XAML layout files.

## 🛠 Troubleshooting Common Issues

- If the project fails to load, check that your project uses UWP and .NET 9 or higher.

- Ensure Visual Studio and .NET SDK are correctly installed on your system.

- Restart ModernUwpDesigner if it crashes or freezes.

- Check file permissions if you cannot save changes.

- For display issues, update your graphics drivers.

- Visit the GitHub page to report bugs or request help.
